# 🍔 Food Delivery Web Application

A full-stack online food delivery web application developed using Java JEE technologies. The application allows customers to browse restaurants, order food online, and manage their orders through a simple and responsive web interface.

---

## 📌 Project Overview

This project is built as a learning project to understand Java Enterprise Edition (JEE) web development using Servlets, JSP, JDBC, and MySQL. It follows the MVC architecture and demonstrates CRUD operations, session management, authentication, and database integration.

---

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/FoodDeliveryApp.git
```

### Import the Project

1. Open Eclipse IDE.
2. Import the project.
3. Configure Apache Tomcat.
4. Create the MySQL database.
5. Execute `schema.sql`.
6. Execute `sample_data.sql`.
7. Update database credentials in the JDBC utility class.
8. Run the project on Tomcat.
